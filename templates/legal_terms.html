شروط الاستخدام

آخر تحديث: أبريل {{ current_year }}{% if college_name %} — {{ college_name }}{% endif %}

باستخدامك لمنصة تعاون فإنك تقر بأنك قرأت هذه الشروط ووافقت عليها. إن لم توافق، يُرجى عدم استخدام المنصة.

1. طبيعة الخدمة

توفر المنصة أدوات لتسجيل المستخدمين، وإدارة طلبات التدريب التعاوني، والتواصل ضمن الإطار الأكاديمي والإداري المعتمد{% if college_name %} في {{ college_name }}{% endif %}. الخدمة قابلة للتطوير أو التعديل أو التعليق لأسباب تقنية أو تنظيمية.

2. الأهلية والحسابات

  • يُفترض أن تكون البيانات المقدّمة عند التسجيل صحيحة ومحدّثة.
  • أنت مسؤول عن سرية بيانات الدخول وأي نشاط يتم عبر حسابك.
  • يُحظر إنشاء حسابات باسم غيرك أو انتحال شخصية جهة أو فرد.
  • للمشغّل حق تعليق أو إنهاء الحساب عند مخالفة الشروط أو عند متطلبات أكاديمية/نظامية.

3. الاستخدام المقبول

يُمنع استخدام المنصة لأي من الأمور التالية:

  • نشر محتوى غير قانوني أو مسيء أو تمييزي، أو ينتهك حقوق الغير.
  • محاولة اختراق الأنظمة، أو إرهاق الخوادم، أو استخراج بيانات بشكل غير مصرح.
  • إرسال رسائل مزعجة أو خادعة للمستخدمين أو الإدارة.
  • استخدام المنصة لأغراض تجارية أو سياسية خارج نطاق التدريب التعاوني المعتمد.

4. المحتوى والملفات

المحتوى الذي ترفعه (مثل السيرة الذاتية أو المستندات) يبقى ملكك، وتمنح المنصة ترخيصًا تشغيليًا لعرضه أو تمريره للجهات ذات الصلة ضمن سير عمل التدريب. يجب ألا يخرق الملفات حقوق ملكية فكرية أو أسرار تجارية لغيرك.

5. العلاقة بين الأطراف

المنصة وسيط تنظيمي وتقني؛ العلاقة التعاقدية أو التدريبية بين الطالب والجهة تخضع لاتفاقيات الجهة والأنظمة الجامعية. لا تضمن المنصة حصول أي طرف على فرصة تدريب محددة.

6. الملكية الفكرية

تصميم المنصة، والشعارات، والنصوص التعريفية الرسمية، محمية. لا يجوز نسخها أو إعادة استخدامها لأغراض تجارية دون إذن كتابي.

7. إخلاء مسؤولية محدود

تُقدَّم الخدمة «كما هي» ضمن الإمكانات التقنية المتاحة. قد تحدث انقطاعات أو أخطاء؛ جرى تفصيل حدود المسؤولية في صفحة إخلاء المسؤولية.

8. التعديل على الشروط

يجوز تحديث هذه الشروط. يُنشر التحديث على هذه الصفحة، ويُفضّل مراجعتها دوريًا. الاستمرار في الاستخدام بعد التعديل يُعد قبولًا للشروط المحدثة ما لم يُنص على خلاف ذلك.

9. القانون والنزاعات

تُفسر هذه الشروط وفق الأنظمة المعمول بها في المملكة العربية السعودية، وتختص المحاكم أو الجهات المختصة في نطاق اختصاصها.

10. التواصل

للاستفسارات المتعلقة بهذه الشروط، يُرجى التواصل عبر القنوات الرسمية لـ{% if college_name %} {{ college_name }}{% else %} الجهة المشغّلة للمنصة{% endif %}.
